# A steady-state visual evoked potential (SSVEP)-based BCI dataset in children and adolescents

[![DOI](https://www.zenodo.org/badge/DOI/10.5281/zenodo.19440996.svg)](https://doi.org/10.5281/zenodo.19440996)

Crawled from Zenodo

## Description

This dataset contains pediatric EEG data recorded during an SSVEP-based BCI experiment that implemented an SSVEP stimulus personalization pipeline. Data from 47 neurotypical participants aged 5-18 years is included in this repository. This data is made publicly available with the aim of promoting the development of signal processing methods specific to pediatric SSVEP-based BCI data.
